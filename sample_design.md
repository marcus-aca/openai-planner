# Project Design: Personal Finance Dashboard

## Goal
Build a simple web app that aggregates transactions, categorizes spending, and provides monthly budgeting insights for individuals.

## Users
- Individuals tracking personal spending.
- Users who want a quick snapshot of monthly budgets.

## Core Features
- Import transactions from CSV.
- Auto-categorize transactions with editable categories.
- Budget creation per category.
- Dashboard view with monthly summary and charts.

## Constraints
- MVP should be achievable in ~4 weeks.
- Single-user, local-only data storage is acceptable.

## Non-goals
- Bank integrations.
- Multi-user accounts.
- Mobile app.

## Success Criteria
- Users can upload a CSV and see categorized transactions.
- Users can set monthly budgets and see current progress.
- Dashboard renders a monthly summary in under 2 seconds.
